# Front-End learning

This repository contains a simple **Front-End web project** created using basic web technologies.

## Technologies Used

* HTML
* CSS
  

## About

This project is built to practice **front-end development skills** and understand how to design simple and responsive web pages.

## Features

* Clean and simple UI
* Basic styling with CSS

## How to Run

1. Download or clone the repository
2. Open the project folder
3. Run the `index.html` file in your browser

⭐ A small project to practice and improve front-end development skills.
